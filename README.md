## Projects

### 01 · Maize Yield Prediction in Kenya Using Machine Learning
**[View project](01_kenya_yield_modelling/README.md)**

Sub-county level maize yield modelling across Trans Nzoia and Uasin Gishu counties 
in western Kenya. Integrates CHIRPS climate data, MODIS vegetation indices, and 
SoilGrids soil properties. Compares six machine learning models (Ridge, LASSO, 
PLSR, RF, GBM, SVM) under a temporal hold-out validation framework. Completed 
independently in R and Google Earth Engine.

`R` `Google Earth Engine` `QGIS` `Random Forest` `SVM` `Ridge Regression` `PCA`
